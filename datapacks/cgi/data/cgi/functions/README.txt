CGI README
==========

Scoreboard objectives and uses
data (dummy) misc. data storage
time (dummy) timers
status (dummy) player states, etc.
UID (dummy) holds UIDs to link players and other entities

Scoreboard dataholders and uses
"gameState" (data) holds state id of the current game state
"gameTime" (data) holds timers for running game
"playerCount" (data) number of living players
"random" (data) where random numbers are generated to
"operand" (data) arbitrary scoreboard holder for doing operations
"i" (data) arbitrary scoreboard holder for doing operations
